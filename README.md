\# Procedural Maze Generator with Pathfinding (Unity)



This project is a procedural maze generator built in Unity, combined with a grid-based pathfinding system.

It was developed as a portfolio project with a focus on clean architecture and clear documentation.



\## Features

\- Procedural maze generation

\- Grid-based maze structure

\- Pathfinding visualization

\- Clean and organized code



\## Technologies

\- Unity

\- C#

\- Pathfinding

\- OOP
